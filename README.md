# Portafolio Personal

Réplico la landing page de Nicolás Dev utilizando Astro, Tailwind CSS y GSAP.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Tecnologías

- [Astro](https://astro.build)
- [Tailwind CSS](https://tailwindcss.com)
- [GSAP](https://greensock.com/gsap/)
